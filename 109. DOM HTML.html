JavaScript HTML DOM - Changing HTML

The HTML DOM allows JavaScript to change the content of HTML elements.

Changing HTML Content

The easiest way to modify the content of a HTML element is by using the innerHTML property.
To change the content of a HTML element, use this syntax.

document.getElementById(id).innerHTML = new HTML


This example changes the content of a <p> element.
Example:-

Hello World!


The above paragraph was changed by a script.
Example Explained:-
  • The HTML document above contains a <p> element with id="demo1"
  • We use the HTML DOM to get the element with id="demo1"
  • A JavaScript changes the content (innerHTML) of that element to "New Text!"

This example changes the content of an <h1> element.
Example:-

Old Heading


JavaScript changed above "Old Heading" to "New Heading".
Example Explained:-
  • The HTML document above contains a <h1> element with id="demo2"
  • We use the HTML DOM to get the element with id="demo2"
  • A JavaScript changes the content (innerHTML) of that element to "New Heading"


Changing the Value of an Attribute

To change the value of a HTML attribute, use this syntax.

document.getElementById(id).attribute = new value


This example changes the value of the src attribute of an <img> element.
Example:-

The original image was "Image2.gif", but the script changed it to "Image3.gif".
Example Explained:-
  • The HTML document above contains an <img> element with id="demo3"
  • We use the HTML DOM to get the element with id="demo3"
  • A JavaScript changes the src attribute of that element from "Image2.gif" to "Image3.gif"
