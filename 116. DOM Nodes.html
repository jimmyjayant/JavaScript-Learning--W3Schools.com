Adding and Removing Nodes (HTML Elements)

Creating New HTML Elements (Nodes)

To add a new element to the HTML DOM, you must create the element (element node) first, and then append it to an existing element.
Example:-

This is a paragraph.

This is another paragraph.

Example Explained

This code creates a new <p> element.

const para1 = document.createElement("p");


To add text to the <p> element, you must create a text node first. This code creates a text node.

const node1 = document.createTextNode("This is a new paragraph.");


Then you must append the text node to the <p> element.

para1.appendChild(node1);


Finally you must append the new element to an existing element.
This code finds an existing element.

const element1 = document.getElementById("demo1");


This code appends the new element to the existing element.

element1.appendChild(para1);

Removing Existing HTML Elements

To remove a HTML element, use the remove() method.
Example:-

This is another paragraph.




Example Explained:-
The HTML document contains a <div> element with two child nodes (two <p> elements).

<div>
<p id="demo7">This is a paragraph.</p>
<p id="demo8">This is another paragraph.</p>
</div>


Find the element you want to remove.

const element2 = document.getElementById("demo7");


Then execute the remove() method on that element.

element2.remove();


The remove() method does not work in older browsers, see the example below on how to use removeChild() instead.

Example Explained

This HTML document contains a <div> element with two child nodes (two <p> elements).

<div id="demo9">
<p id="demo10">This is a paragraph.</p>
<p id="demo11">This is another paragraph.</p>
</div>


Find the element with id="demo9".

const parent1 = document.getElementById("demo9");


Find the <p> element with id="demo10".

const child2 = document.getElementById("demo10");


Remove the child from the parent.

parent1.removeChild(child2);


Here is a common workaround: Find the child you want to remove, and use its parentNode property to find the parent.

const child2 = document.getElementById("demo10");
child2.parentNode.removeChild(child2);
